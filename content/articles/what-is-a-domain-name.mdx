A domain name is the human-readable address used to reach a website, such as `example.com`. It exists because numbers are hard to remember, and most people would rather type words than raw IP addresses.

## Why Domain Names Matter

Computers still rely on numerical addresses behind the scenes, but domain names make the web usable for humans. They act like labels that point to the correct destination.

That is why your browser can open a site when you type a name instead of a long string of digits.

## A Domain Name Is Not the Website Itself

People often talk as if a domain and a website are the same thing, but they are separate. The domain is the address. The website is the content and systems hosted somewhere behind that address.

You can move a website to a different server and keep the same domain name, or point one domain to a new destination.

## Where DNS Fits In

DNS is the system that translates domain names into the numerical addresses computers actually use. Without DNS, domain names would not do much on their own.

That is why domain names and DNS are closely linked but not identical. One is the readable label, and the other is the lookup system.
